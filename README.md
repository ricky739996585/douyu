# 						**斗鱼直播项目**

## 1.弹幕抓取项目

​	该项目主要的作用是抓取直播间的弹幕信息，然后通过分析弹幕信息、礼物信息来进行一系列的操作。

1. 连接弹幕服务器，有重连机制保持持续连接。

2. 获取弹幕信息，主要抓取指令性操作的弹幕信息。如点播电影、查看用户积分、签到等操作命令。

   命令格式如下：

   ```
   1.  #查询  
   2.  #打卡  
   3.  @电影名称-分数  如：@千与千寻-100
   ```

3. 获取礼物信息，然后对用户的积分进行修改，最后修改展示用于积分情况变化的文件内容。
